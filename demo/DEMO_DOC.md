# AlphaInvestPro 演示文档（简版）

> 本文档与自动录制的演示视频（`demo/record_demo.mjs`）一一对应，可作为讲解词或独立阅读材料。
> 声明：AlphaInvestPro 仅供研究使用，不构成投资建议；AI 输出与估值假设可能有误，不承诺任何盈利。

## 1. 项目简介

AlphaInvestPro 是一款**仅供研究**的 AI 智能投资分析工作台。输入一个股票代码，即可获得一份结构化的投资备忘录：商业质量、护城河、估值、风险审视、多空观点与最终结论。核心设计原则：

- 所有数字来自**确定性计算**，AI 只负责叙述，绝不编造数据；
- 分析过程完全可视化，拒绝黑盒式聊天机器人；
- 不含任何交易执行、券商接入或自动化下单功能。

## 2. 使用场景说明

- **个人投资者**：希望用分析师清单式的纪律做股票研究，而不是碎片化的 AI 聊天；
- **投研初学者 / 商科学生**：需要一个把"质量、估值、风险、结论"分开的结构化框架；
- **产品经理 / 创业者**：需要一个可以直接演示的投资分析产品原型；
- **AI 应用开发者与研究者**：探索"多步骤智能体 + 确定性计算 + 大模型叙述"的工程范式。

## 3. 核心功能演示

演示路径（与视频一致）：

1. **仪表盘**：报告数量、观察列表、数据源与大模型运行模式一目了然；
2. **新建分析**：输入股票代码，四种模式可选——快速筛查 / 完整备忘录 / 风险审视 / 估值检查，报告语言支持中英文；
3. **可视化 Agent 时间线**：输入规范化 → 公司识别 → 快速筛查 → 商业质量 → 护城河 → 管理层与资本配置 → 估值 → 逆向/风险审视 → 多空观点 → 最终备忘录，每一步实时推送（SSE）；
4. **观察列表**：一键收藏标的，持续跟踪最近结论；
5. **报告历史**：所有备忘录自动归档，随时回看。

## 4. AI 能力展示

- **多步骤智能体编排**：十个分析步骤按模式自动裁剪、顺序执行、逐步可见；
- **大模型叙述层（DeepSeek）**：只基于已验证的结构化数据撰写解读，从机制上杜绝"AI 编造数字"；
- **确定性打分引擎**：快速筛查清单、商业质量与护城河 0-10 分、合理价值区间全部由算法计算；
- **优雅降级**：未配置密钥时自动切换为明确标注的「模拟数据 / 模拟 LLM 输出」模式，流程照常演示。

## 5. 输入、处理过程和输出结果

| 环节 | 内容 |
| --- | --- |
| **输入** | 一个股票代码 + 分析模式 + 报告语言 |
| **处理** | 数据网关拉取行情/基本面/新闻 → 规范化并记录来源与时间戳 → 确定性打分与估值 → 大模型叙述 → 汇总生成结构化 JSON |
| **输出** | 一份完整投资报告：最终结论与置信度、筛查清单、质量/护城河评分、低-基准-高估值区间与**全透明假设表**、主要风险与"逆向提问"、多空观点对照、Markdown 备忘录（支持一键导出 MD/JSON）、证据与数据来源清单 |

## 6. 项目亮点说明

1. **过程透明**：每一步分析实时可见，结论可追溯；
2. **数字可审计**：每条估值假设都标注名称、取值与来源；每条数据都有提供方与时间戳留痕；
3. **中英双语**：整个界面与报告内容支持简体中文 / English 即时切换，不刷新、不丢状态；
4. **诚实降级**：模拟数据与模拟 AI 输出永远显著标注，绝不冒充真实数据;
5. **安全边界**：仅研究、无交易执行，API 密钥只存在于后端环境变量，前端永不可见；
6. **开箱即用**：无需任何密钥即可本地运行完整演示，配好密钥即升级为真实数据与真实 AI 叙述。
